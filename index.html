Welcome to Scramble.

The game where you unscramble letters to make words.

Introduction

Scramble is a text-based console game. So you must open the web console to play.

Instructions

Once you start the game, you will be given a scrambled word.

If you correctly guess the word, you will receive a point. If you guess incorrectly you will receive a strike. You can also pass on a word.

Commands

To start a new game use start().

To make guess use guess('word').

To skip a word use pass().

To show the instructions use help().
